# Guess-Number_Game 
<p>This is a simple game developed using vanilla js.
Key feature includes Start & check buttons, color change function.
You're only required to enter numbers between 1-5 and click "Check".</p>
<p>If the number entered matches the random number generated by the App, it displays "Correct",
else, it'll display "Opss you're wrong".</p>
<p>To restart the game click "Start" button.</p>
